About Servrite

Servrite is built for bars, hotels,restaurants

and eateries of all kind.

It is designed to be used within these

establishments by waiters/waitress and

barmen/barwomen to take customer's

orders.

It was built in the year 2022


Servrite eliminates the issue of delayed

orders and serving wrong orders and helps

the establishment makes better,informed decisions

on the product



HOW TO USE

To use servrite is very simple please follow the instructions

accordingly to ensure smooth use.

*Login

*Click on the select a menu buuton

*Input the quantity

*Click on Add order

*Click on place order

*Then finnaly select a payment option


Further explanation of how to use ServRite:

After each menu is slected and the quantity

has been inputed, click on the "Add order"

button then select select another menu and

repeat the process for as many orders you are selecting.

To delete an order select the menu in the summary box

and the press the "Delete" button.
